## 🚀 Run Locally in Seconds

Since DevToolbox is a static page index, you can run it locally in seconds:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/bruh74392-collab/dev-toolbox.git
   cd dev-toolbox
   ```

2. **Open in browser**:
   Simply open `index.html` directly in your browser:
   ```bash
   # On macOS
   open index.html
   ```

3. **Or run with a local server (Optional)**:
   ```bash
   python -m http.server 8000
   ```
   Then open `http://localhost:8000`.
